# utils 内部工具函数
## jwt规范
官方标准字段
| 字段  | 含义            | 是否推荐 |
| --- | ------------- | ---- |
| iss | token签发者      | 推荐   |
| sub | token主体（用户ID） | 强烈推荐 |
| aud | token接收方      | 推荐   |
| exp | 过期时间          | 必须   |
| nbf | 生效时间          | 可选   |
| iat | 签发时间          | 强烈推荐 |
| jti | token唯一ID     | 强烈推荐 |
